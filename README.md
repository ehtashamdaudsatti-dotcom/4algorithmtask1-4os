# Operating System Scheduling Algorithms

This repository contains implementations of four CPU Scheduling Algorithms developed using HTML, CSS, JavaScript, Node.js, and Express.js.

## Projects Included

### 1. Round Robin Scheduling
- Time Quantum based scheduling
- Gantt Chart Generation
- Node.js Backend
- HTML/CSS/JavaScript Frontend

### 2. Shortest Job First (SJF)
- Non-Preemptive SJF Scheduling
- Sorts processes by Burst Time
- Displays execution order
- Gantt Chart Visualization

### 3. First Come First Serve (FCFS)
- Executes processes in arrival order
- Simple CPU Scheduling
- Gantt Chart Output
- Interactive User Interface

### 4. Priority Scheduling
- Executes processes based on Priority
- Lower Priority Number = Higher Priority
- Displays Process Priority
- Gantt Chart Visualization

## Installation

Clone the repository

```bash
git clone https://github.com/ehtashamdaudsatti-dotcom/4algorithmtask1-4os.git
```

Go to any project backend

```bash
cd roundrobin/backend
```

Install dependencies

```bash
npm install
```

Start backend server

```bash
npm start
```

Open the frontend in your browser and use the simulator.
